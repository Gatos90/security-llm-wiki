# EPSS

The Exploit Prediction Scoring System estimates the probability that a vulnerability will be exploited in the wild over a near-term time window. It complements, but does not replace, severity scoring.

Use EPSS as one public signal among several:

- Known exploitation reports.
- CISA Known Exploited Vulnerabilities catalog membership.
- Public exploit code or weaponized tooling.
- Internet-facing exposure patterns.
- Availability and operational cost of mitigation.
