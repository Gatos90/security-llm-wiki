# Maven Security

Maven vulnerability records should use group ID, artifact ID, and version coordinates. Some Java vulnerabilities affect frameworks, plugins, or application server integrations rather than a single library.

Useful public fields:

- `groupId`
- `artifactId`
- Affected versions.
- Fixed versions.
- Public advisory references.
- Notes about shaded or bundled dependencies when publicly documented.
