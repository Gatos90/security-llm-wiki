# Container Security

Container vulnerability information may come from operating system packages, language packages, image metadata, or application components.

Public-safe normalization should distinguish:

- Base image vulnerabilities.
- Application dependency vulnerabilities.
- Runtime configuration risks.
- Fixed image tags or digest references when public.
- Vendor image release notes.
